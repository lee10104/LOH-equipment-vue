# 로오히 장비 세팅 도우미

## 실제 사용을 위해 남은 할 일
1. 계산 결과 정렬 기능 추가
2. 계산 진행 상황 시각화

## 우선순위는 떨어지지만 언젠간 추가 예정인 기능
- 장비 삭제 기능 추가
- 장비 요약에 부위 정보 추가
- 장비 입력창 개선
  - validation 추가
  - 별 갯수, 강화 단계 설정 기능 추가
  - 장비 주옵션은 강화 단계 && 스탯에 따라 자동으로 입력
  - 장비 부옵션은 4개 모두 있는 것을 default로
- 장비 정보를 툴팁으로 보여주기
- 이상/미만 이외의 조건
  - 선택한 장비 무조건 포함/제외
  - 세트 맞추기
- 각종 단축키 추가
- css 개선
- 모바일 대응(이건 실험해보고 계산이 어렵겠다 싶으면 관둘 예정)
